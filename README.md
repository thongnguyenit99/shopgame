# shopgame chạy trên PHP version 7.3.3 và port:8000

### Bài tập này em cho project chạy trên Xampp và viết trên Visual Studio Code
#### Bài làm có sử dụng kỹ thuật Ajax thuần khi thêm sản phẩm và tính giá tiền và file .htaccess
##### DB tên:shopgame.sql lưu trong thư mục models .Thêm sản phẩm bằng file CSV thì ở file .csv có các cột: id_url,tên sản phẩm,chi tiết,giá,giá khuyến mãi,hình,hãng,số lượng và cột trạng thái cho sản phẩm(xóa hay chưa xóa)
##### Đăng nhập vào trang Admin thì sử dụng tài khoản admin và password admin ,có sử dụng session để lưu phiên là chỉ cho khi đăng nhập mới vào trang admin.có logout sẽ hủy session phiên đó.Giỏ hàng có thể edit và xóa sản phẩm.
